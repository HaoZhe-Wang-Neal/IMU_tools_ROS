Open source IMU data processing tools for ROS

### ROS 生态 IMU 数据处理工具集 
    基于 C++/Python 开发符合 ROS 规范的 IMU 数据处理工具包，已落地 Bag 转 CSV、IMU 数据清洗、VQF 轨迹算法等功能，兼具高性能与易用性，支持模块化调用与全参数配置化，后续将持续迭代完善。
    遵循 “模块化、可插拔、易扩展” 设计原则，将各核心功能封装为独立类 / 函数，无跨模块耦合，支持单独 #include/.import 调用，可灵活适配不同数据处理链路的集成需求。
    采用全参数配置化设计，基于 YAML 文件统一管理所有关键参数；通过参数加载函数实现配置灵活切换，适配多场景 ROS 下 IMU 数据处理需求，核心处理逻辑可移植至其他业务场景。
    全功能模块均遵循 ROS 标准规范实现 C++/Python 双语言版本：C++ 版本重视大规模数据处理、算法运行效率；Python 版本降低算法调试、参数调优及二次开发成本，重视易用性，所有版本均可集成至 ROS 工作空间。
    工具包处于长期维护迭代阶段，计划持续完善并构建 ROS 生态下 “清洗 - 补偿 - 融合” 的完整 IMU 数据处理体系。
